# Frontend 수정사항
1. Main page : 왼쪽 하단 '일과 추가'를 '일정 추가'로 변경, 근무 피로도를 계산한 수치도 Main page에서 확인할 수 있었으면 좋겠음.
2. 학습 로드맵 page : 학습 로드맵 대로 실천했을 때, 이것이 시간이 지나면 자동으로 체크가 되는건지 직접 하는건지 구분이 어려움. 직접 체크표시를 통해 완료했다는 것을 알려줬으면 좋겠음.
3. 추천 page : 상단 버튼 색깔을 좀 연한 검정색으로 했으면 좋겠음. 눈이 아픔.
4. 내 정보 page : Main page에서는 사용자 이름이 명확히 표시가 되었는데, 내정보 page에는 undefined로 표시가 됨. 자기개발 설정 수정 버튼은 자기개발 설정 칸 안의 오른쪽 상단에 있었으면 좋겠음. 
5. 알림 page : fronend만 구성된 것 같은데, 이것도 백엔드와 연겷하여 활성화 해야함.
6. 설정 page : 알림 부분의 on/off 토글의 'O'의 위치가 오른쪽으로 치우쳐져 있음. 
7. 목표 관리 page : 현재 목표를 생성하면 바로 일시 중지가 됨. 목표를 누르면 세부 목표 내용으로 넘어가지도 않음.
8. 일정 추가 page : 일정 추가 페이지의 시작시간과 종료시간은 근무의 경우 다음날로 넘어가는 경우가 많음 (Ex. 4월 2일 오후 6시 ~ 4월 3일 오전 8시) 하지만 다음날로 넘어가지 못하게 되있음

# 미구현된 Frontend 사항
1. 프로필 수정 page : 프로필 수정과 자기개발 설정 수정은 다른거임. 프로필 정보를 수정하는 화면이 없음.
2. 일정 수정 page : 일정 수정 page가 기존에 있었으나 없어진 것으로 보임.
3. 비밀번호 변경 page : 비밀번호 변경 페이지가 없음

# Backend 수정사항
1. 강의 추천 : 강의가 지금 안뜸. 목표 기반 강의 추천받기를 누르면 'Not Found'로 넘어감. 현재 초기 MVP 모델로서는 K-MOOC API를 직접 연결하기 보다는, 직접 데이터를 미리 만들어서 강의를 저장하고, 그것을 Frontend로 보여줘야 할 것 같음. 
2. AI 채팅 : 배포해서 그런지 AI가 연결되어 있지 않은 것 같음. API를 분명 환경변수에 넣었음.
3. 목표 관리 : 목표를 생성하면 바로 일시중지로 감. 그리고 재개 버튼을 누르고 다른 페이지 갔다오면 다시 일시중지가 됨. 목표 삭제도 안됨.

# 미구현된 Backend 사항
1. 알림 : 특수 조건일 때 알림을 오게 하는 방법이 좋을 것 같음.
2. 목표 기반 강의 추천받기 : 이미 내장되어 있는 강의 목록으로 목표 기반 강의를 추천받아야 할 것으로 보임.
3. 로드맵이 너무 단순함. 
4. 설정의 알림 기능 : 푸시 알림, 일정 알림, 강의 추천 알림, 목표 리마인더가 모두 미구현됨.
5. 비밀번호 변경 : 비밀번호 변경이 현재 구현이 안됨
6. 강의 목록 : 강의 목록을 보여주는 것도 좋을 것 같음. frontend도 해당사항임.
